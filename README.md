##### 片元着色器代码会在GPU的片元着色器单元执行
##### 在WebGL渲染管线流程中，或者说GPU的渲染流程中，顶点着色器代码先执行处理顶点，得到一系列片元，然后再执行片元着色器代码处理片元。

##### webGL系统中的一块内存中区域，可一次性填充大量顶点数据并保存，供后续顶点着色器使用

##### 使图形和几何学的计算在投影空间中成为可能。齐次坐标是用N+1个数来表示N维坐标的一种方式。
